# Horizon Website

A small static landing page built with HTML and Tailwind CSS via `@tailwindcss/browser`.

## Project Structure

- `public/index.html` — main page markup
- `public/styles.css` — custom site styles
- `public/images/` — image assets used on the site

## Preview

Open `public/index.html` in your browser to view the site.

## Notes

- The project uses the Tailwind Browser CDN for utility classes.
- You can edit the page content and images in the `public` folder.
